「Webコンテンツ作成」の成果

本テーマについて

HTML,CSS,Javascriptを用いてWEB制作をしていく。またその中で手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できるようになることが本テーマの目的である。

数当てゲーム

コンピュータが乱数で1〜10の値を1つ決め、ユーザがその数字を予想するゲームを制作した。ユーザは最大3回まで数字を入力でき、不正解の場合は答えが入力した数字より大きいか小さいかをヒントとして表示する。3回以内に正解できれば勝ち、できなければ負けとなる仕組みである。制作していて一番苦労した点は、条件分岐をどの順番で設定するかだった。特に4回目以降もゲームが続いてしまう不具合の修正や、比較演算子や条件式の単純な記述ミスによるエラーが多く、思うように動かず時間がかかった。結局コードを見続けるのではなく全体の処理の流れを理解することと休憩を挟むことで小さなミスが減り完成させることができた。

第1回 WWW HTML

学習内容：今回の学習では、HTMLがWebページの構造を記述するための言語であり、要素・タグ・属性によって文書を作成することを学んだ。また、HTML文書全体の基本構成としてDOCTYPE宣言、html、head、body要素の役割を理解した。さらに、空要素や属性の使い方、特殊文字の記述方法、コメントの書き方について学び、正しいHTML文書を作成する方法を身につけた。しかし理解しきれていない部分もありどんどん新しい内容も増えていくためしっかり復習しておきたい。
演習の成果：演習1-4：HTMLのさまざまな要素

第2回 CSSによる Web ページの装飾

学習内容：今回の学習では、CSSはHTMLの見た目や装飾を設定する言語であることを学んだ。まず基本的な記述方法について学び。CSSではセレクタで対象の要素を指定し、colorやfont-weightなどのプロパティに値を設定してデザインを変更する。また、class属性を利用することで特定の要素だけを装飾できることや、リンクやリストの見た目も細かく設定できることを理解した。課題などで自分の作りたいデザインを再現できるようこれからも学んでいきたい。
演習の成果：演習2-3：ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

学習内容：今回の学習では、JavaScriptとはどういったものか学んだのち開発者ツールのコンソールを利用して、console.log()による出力やエラーメッセージの確認、簡単なプログラムの実行・デバッグを行いました。前回よりわからない単語が増えついていけるか不安になりましたが頑張ります。
演習の成果：演習3-6: オブジェクトの参照

第4回 DOM と JavaScript

学習内容：今回の授業では、JavaScriptを用いて「ボタンを押したらページの内容が変化する」仕組みについて学んだ。4-2~4-6まではなんとか理解できていたがそれ以降のイベント処理についての話から置いていかれるようになったため家で自習した。結局のところ4-7にあるコードの１行１行が何をしているのか分解して理解するという当たり前のことに行き着いた。正直横文字ばかりで一度聞いたことを戻って確認することの繰り返しに疲れたがなんとかくらいつけている気がする。
演習の成果：演習4-10: 足し算ページ

第5回 イベント処理

学習内容：今回の授業では、DOMを利用しJavaScriptからWebページの内容を動的に変更する方法を学んだ。前回学習した要素の検索やテキストの取得・変更に加え、今回は要素の新規作成、属性の変更、要素の削除を行う方法を実践した。createElement()で新しい要素を作成し、insertAdjacentElement()でページ内の指定した位置に追加することで、リストやリンクなどを表示できることを理解した。また、setAttribute()を用いて画像のsrc属性やリンクのhref属性を設定し、ページの内容を変更する方法も学んだ。演習では、ボタンのクリックイベントに関数を登録し、大学のキャンパス住所や工学部の学科一覧をオブジェクトや配列から取得してページ上へ表示するプログラムを作成した。前回よりはすんなり理解ができた気がするが5-7でつまづいて時間がかかった。最終課題も目前なので全体的に復習をしようと考えた。
演習の成果：演習5-2: DOM演習その2
