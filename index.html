各授業回の学習事項

第1回 WWW HTML

学習内容：今回の学習では、HTMLがWebページの構造を記述するための言語であり、要素・タグ・属性によって文書を作成することを学んだ。また、HTML文書全体の基本構成としてDOCTYPE宣言、html、head、body要素の役割を理解した。さらに、空要素や属性の使い方、特殊文字の記述方法、コメントの書き方について学び、正しいHTML文書を作成する方法を身につけた。しかし理解しきれていない部分もありどんどん新しい内容も増えていくためしっかり復習しておきたい。
演習の成果：演習1-4：HTMLのさまざまな要素

第2回 CSSによる Web ページの装飾

学習内容：今回の学習では、CSSはHTMLの見た目や装飾を設定する言語であることを学んだ。まず基本的な記述方法について学び。CSSではセレクタで対象の要素を指定し、colorやfont-weightなどのプロパティに値を設定してデザインを変更する。また、class属性を利用することで特定の要素だけを装飾できることや、リンクやリストの見た目も細かく設定できることを理解した。課題などで自分の作りたいデザインを再現できるようこれからも学んでいきたい。
演習の成果：演習2-3：ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

学習内容：今回の学習では、JavaScriptとはどういったものか学んだのち開発者ツールのコンソールを利用して、console.log()による出力やエラーメッセージの確認、簡単なプログラムの実行・デバッグを行いました。前回よりわからない単語が増えついていけるか不安になりましたが頑張ります。
演習の成果：演習3-6: オブジェクトの参照
